## 🎤 想定される質問と回答

### Q1: セキュリティは大丈夫ですか？
**A:**
```
はい、以下のセキュリティ対策を実施しています：

- HTTPS通信（SSL/TLS暗号化）
- アップロードファイルのバリデーション
- セッション管理によるデータ保護
- 生成されたPDFは一時的に保存され、定期的に削除

今後、ユーザー認証機能を追加する際は、
さらに強固なセキュリティ対策を実装します。
```

### Q2: 大量のデータも処理できますか？
**A:**
```
現在のMVPでは、1ファイルあたり最大10MBまで対応しています。
これは通常の受注データであれば十分なサイズです。

今後、一括処理機能を追加する際は、
より大規模なデータ処理にも対応する予定です。
```

### Q3: 既存のシステムと連携できますか？
**A:**
```
はい、CSV形式でデータをエクスポートできるシステムであれば、
すぐに連携可能です。

また、今後API機能を追加することで、
既存システムから直接データを送信できるようになります。
```

### Q4: カスタマイズは可能ですか？
**A:**
```
はい、可能です。

現在でも設定ファイルを編集することで、
レイアウトやフォーマットをカスタマイズできます。

今後、管理画面からGUIで設定できる機能も追加予定です。
```

### Q5: コストはどのくらいですか？
**A:**
```
現在のホスティングコストは月額約$7です。

本番運用時は、利用規模に応じて
適切なプランを選択できます。

また、オンプレミス環境での運用も可能です。
```

### Q6: スマートフォンでも使えますか？
**A:**
```
現在のMVPはPCブラウザでの利用を想定していますが、
スマートフォンのブラウザでも基本的な操作は可能です。
```

## 🚨 トラブルシューティング

### 問題1: 初回アクセスが遅い
**原因:** Renderの無料プランはスリープする
**対策:** デモ5分前に一度アクセスしてウォームアップ

### 問題2: PDFが文字化けする
**原因:** フォント設定の問題
**対策:** 現在は修正済み（DejaVu Sans Condensed使用）

### 問題3: ファイルアップロードが失敗する
**原因:** ファイルサイズまたは形式の問題
**対策:** サンプルCSVを使用（動作確認済み）

### 問題4: ダウンロードが開始されない
**原因:** ブラウザのポップアップブロック
**対策:** ポップアップを許可するよう案内
