## 📝 複数ページ機能の技術説明

### 実装方法

**トーク（技術的な質問があった場合）:**
```
ページ分割は、mPDFライブラリの機能を活用しています。

【分割ロジック】
1. コンテンツの高さを計算
2. A4用紙の印刷可能領域を考慮
3. アイテム単位で分割判定
4. 自動的に改ページを挿入

【最適化】
- アイテムブロックは分割しない
- パーツ一覧表は適切な位置で分割
- ヘッダー・フッターは各ページに配置

これにより、印刷時の見やすさと
作業効率を両立しています。
```
